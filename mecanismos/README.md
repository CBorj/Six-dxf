# Mecanismos

Archivos DXF de piezas mecánicas y mecanismos para corte CNC, láser o plasma.

## Contenido

Esta carpeta contiene diseños de:
- Engranajes
- Poleas y correas
- Articulaciones
- Bisagras
- Componentes mecánicos

## Formato

Todos los archivos deben estar en formato DXF compatible con máquinas de corte CNC.
