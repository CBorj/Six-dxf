# Plantillas

Archivos DXF de plantillas base y guías para corte CNC, láser o plasma.

## Contenido

Esta carpeta contiene:
- Plantillas base para proyectos
- Guías de diseño
- Formatos estándar
- Marcos de referencia
- Templates reutilizables

## Formato

Todos los archivos deben estar en formato DXF compatible con máquinas de corte CNC.
