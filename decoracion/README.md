# Decoración

Archivos DXF de elementos decorativos para corte CNC, láser o plasma.

## Contenido

Esta carpeta contiene diseños decorativos tales como:
- Paneles decorativos
- Ornamentos
- Arte de pared
- Elementos de diseño interior
- Figuras decorativas

## Formato

Todos los archivos deben estar en formato DXF compatible con máquinas de corte CNC.
