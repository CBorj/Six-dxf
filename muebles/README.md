# Muebles

Archivos DXF de diseños de muebles para corte CNC, láser o plasma.

## Contenido

Esta carpeta contiene diseños de:
- Mesas y sillas
- Estanterías
- Muebles modulares
- Piezas de ensamblaje
- Componentes estructurales

## Formato

Todos los archivos deben estar en formato DXF compatible con máquinas de corte CNC.
