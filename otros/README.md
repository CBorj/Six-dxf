# Otros

Archivos DXF de diseños diversos para corte CNC, láser o plasma.

## Contenido

Esta carpeta contiene diseños que no encajan en las categorías principales:
- Proyectos especiales
- Diseños experimentales
- Prototipos
- Otros diseños diversos

## Formato

Todos los archivos deben estar en formato DXF compatible con máquinas de corte CNC.
